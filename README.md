# Проект по математической статистике

Тема: **"Статистика в области кибербезопасности: анализ угроз и обнаружение аномалий"**

В проекте использован датасет **RT-IoT2022** из UCI Machine Learning Repository:
https://archive.ics.uci.edu/dataset/942/rt-iot2022

## Структура

- `src/analyze_rt_iot2022.py` - основной скрипт анализа
- `src/present_markdown.py` - простой консольный презентер Markdown
- `data/raw/rt_iot2022.csv` - локальная копия исходного датасета
- `outputs/tables/` - таблицы с результатами
- `outputs/plots/` - построенные графики
- `methods/` - короткие конспекты по использованным статистическим методам
- `report.md` - готовый отчет
- `defense_notes.md` - шпаргалка к защите

## Как запустить

```bash
source .venv/bin/activate
python src/analyze_rt_iot2022.py
```

## Что делает скрипт

1. Загружает и кэширует датасет.
2. Формирует бинарную переменную:
   - `Attack` - вредоносный трафик
   - `Normal` - штатный трафик
3. Считает описательные статистики.
4. Строит гистограммы, боксплоты и диаграмму рассеивания.
5. Ищет выбросы по межквартильному размаху.
6. Проверяет статистические гипотезы.
7. Строит линейную регрессию.
