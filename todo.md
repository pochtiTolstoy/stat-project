1. Получить основную информацию о данных: количество
наблюдений, количество переменных, типы данных,
количество пропущенных значений. Пропущенные
значения либо заполнить, либо удалить, если данных
достаточно.

2. Вычислить основные статистические характеристики
данных: средние значения, дисперсии, корреляции,
минимумы, максимумы, квартили.

3. Построить гистограммы с графиком функции
распределения, боксплоты и диаграммы рассеивания.

4. Найти выбросы с помощью межквартильного размаха.

5. На основе выбранных данных провести алгоритм
проверки гипотезы о параметрах генеральной совокупности
и виде распределения.

6. Выбрать два признака и построить линейную регрессию.

7. Написать отчет о полученных результатах.
Здесь Вы можете взять данные для своего
проекта:

https://github.com/Yorko/mlcourse.ai/tree/main/data
https://www.kaggle.com/datasets
https://archive.ics.uci.edu/ml/datasets.php
