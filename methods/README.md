# Методы проекта

В этой папке собраны короткие конспекты по методам, которые использовались в проекте.

- `01_iqr_outliers.md` — поиск выбросов методом межквартильного размаха
- `02_welch_t_test.md` — критерий Уэлча для сравнения средних
- `03_dagostino_normality.md` — тест Д'Агостино на нормальность
- `04_linear_regression.md` — линейная регрессия
- `05_correlation.md` — корреляционный анализ

Если преподаватель будет подробно спрашивать про расчеты, лучше открыть нужный файл отдельно и пройтись по нему.
